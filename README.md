# Greentrip Frontend

## Installation

1. Install dependencies:
    ```sh
    npm install
    ```

2. Install Git LFS:
    ```sh
    git lfs install
    ```

3. Start the development server:
    ```sh
    npm run dev
    ```

## Project Structure

- `src/`: Source code of the application
- `public/`: Public assets
